# GitHub Pages - IT Support Technician

This is a **static, optimized** personal site:
**IT Support Technician / Desktop & Field Support**.

## What’s Included
- `index.html` - Accessible, SEO-optimized, fast page.
- `assets/styles.css` - Minimal responsive styling, dark-mode aware.
- `robots.txt` & `sitemap.xml` - Search engine friendliness.
- `404.html` - Friendly not‑found page.
- `.nojekyll` - Ensures GitHub Pages serves files as-is.
- `README.md` - These instructions.

## SEO Notes
- Semantic HTML + JSON-LD `Person` schema.
- Canonical tag + Open Graph + Twitter cards.
- Lightweight (no JS framework), fast load.

## Accessibility
- Skip link, high contrast, keyboard focus styles.
- Respects `prefers-color-scheme` for dark mode.
